My Code Katas
==============

This repository holds a number of kata exercises I used to practice

* TDD
* SOLID Principle
* BDD
* Clean Coding
* Software Design
* Agile
* Pair programming

##
To run the php test,
- install phpunit v6^
- `cd myCodeKatas`
- run as `phpunit <path to test>`
##
To run the Js test,
-`npm install`
- `npm test`
